演習10-13(解答)
===============

問い
----

foldlを使ってreverseを定義せよ。

解説
----

foldlを使う場合には以下のように考える。

* 初期値は何か?
* リストの次の値によって蓄積変数はどう変わるか

reverseについてこれを考えると

* 初期値: []
* 次の値xによって蓄積変数ysは(x : ys)となる

解答
----

    reverse :: [a] -> [a]
    reverse = foldl (\ys x -> x : ys) []

\ys x -> x : ysは(:)の引数を入れ換えただけなので

    reverse = foldl (flip (:)) []
