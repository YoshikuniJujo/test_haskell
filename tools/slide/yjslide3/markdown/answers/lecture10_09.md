演習10-9(解答)
==============

問い
----

再帰を直接使ってfilterを定義せよ。

解説
----

以下のように考える。

* 空リストからは何も選べないので結果も空リストである
* 先頭の要素が条件を満たす場合
    + 残りの要素をfilterしたものに先頭の要素を追加したもの
* 先頭の要素が条件を満たさない場合
    + 残りの要素をfilterしたもの

解答
----

    filter :: (a -> Bool) -> [a] -> [a]
    filter _ [] = []
    filter p (x : xs)
        | p x = x : filter p xs
        | otherwise = filter p xs
