第5回 演習8 (解答)
==================

問い
----

(-1, 1), (1, 1), (1, -1), (-1, -1)を4点とする正方形内に
ランダムな点をうつ。
ランダムの種を引数としてとり、
その点の列を表すリストを作る関数をpointsとする。

pointsの型を決めよ。

考えかた
--------

pointsの入力はランダムの種なので

    StdGen

pointsの出力は点のリストなので

    [(Double, Double)]

解答
----

    points :: StdGen -> [(Double, Double)]
