演習10-8
========

問い
----

unfoldrを使ってmapを定義せよ。

解説
----

以下の3つを考える。2, 3は「引数(x : xs)に対して」ということ。

1. 終了条件: []になれば終了
2. 結果のリストに変まれる値: f x
3. 次の計算にわたす値: xs

頭をけずったリストを次々にわたしながら、
頭の値にfを適用したものを集める関数、ということになる。

    map f = unfoldr $ \lst -> case lst of
        [] -> Nothing
        x : xs -> Just (f x, xs)
