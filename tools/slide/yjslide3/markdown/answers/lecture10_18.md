演習10-18(解答)
===============

問い
----

関数unzipはタプルのリストをそれぞれの要素ごとのリストに分ける。
再帰を直接使って関数unzipを定義せよ。

解説
----

* 空リストなら空リストのペアになる
* リストのはじめの要素以外をunzipしたものがあれば
    + それぞれのリストにペアのそれぞれの値を追加したもの

解答
----

    unzip [] = ([], []_
    unzip ((x, y) : xys) = (x : xs, y : ys)
        where
        (xs, ys) = unzip xys
