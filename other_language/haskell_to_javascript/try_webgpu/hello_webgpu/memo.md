# Texture Formats

[6.3. Texture Formats](https://gpuweb.github.io/gpuweb/#texture-formats)

* r, g, b, a: red, green, blue, alpha
* unorm: unsigned normalized
* snorm: signed normalized
* uint: unsigned int
* sint: signed int
* float: floating point
