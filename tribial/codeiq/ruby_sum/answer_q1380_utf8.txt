【必須】コメント部分をコードに書き替えてください（そのほかの部分は変更しないでください。）

def sum(data) # dataは整数の配列
	data.inject(0) { |s, x| s + x }
end

v = [3, 8, 2, 6, 1, 7, 9]
puts sum(v)

【必須】後輩プログラマAさんにひとことアドバイスをお願いします。

Rubyでは、配列に対してインデックスでアクセスする必要があることはあまりありません。
配列のeachメソッドに対してブロックを与えてやる方法もrubyらしくて良いでしょう。
for式を使えばよりスマートに書けますが、
ループ変数を破壊的に変更するので使わないほうが無難です。

今回のように配列等のコレクションの中身を二引数関数に与えていき、
ひとつの値を作るようなアルゴリズムを「たたみこみ」と呼びます。
Rubyではinjectというメソッドが用意されています。

いずれにしてもRubyでは「ブロック」という仕組みを使うことで、
関数を「ひとつだけ」引数に取る関数をスマートに書くことができます。
ブロックを取る関数は自分で定義することができます。
「ruby yield」で検索してみてください。
